# 🚀 VORAZONE — Company Profile Website

Website company profile untuk **Vorazone**, brand yang fokus pada smart gadgets & electronic accessories untuk meningkatkan kualitas hidup sehari-hari.

---

## 👋 Overview

Vorazone adalah platform digital yang menampilkan produk-produk inovatif dengan pendekatan desain modern, performa cepat, dan pengalaman user yang smooth.

Website ini dibangun sebagai:

* Media branding
* Showcase produk
* Lead generation (form contact)
* Landing page yang scalable & SEO-friendly

---

## ⚙️ Tech Stack

### 🧱 Core Framework

* Astro 6.0.2 — Static Site Generator (SSG)

### 🎨 Styling & UI

* Tailwind CSS
* Lucide Icons
* Google Fonts API

### 🎞 Animation & Interaction

* Framer Motion

### 📩 Form Handling

* Web3Forms

### 🔐 Security

* HSTS (HTTP Strict Transport Security)

### ☁️ Deployment

* Vercel — deployment cepat dengan CDN global & auto CI/CD

---

## 🧩 Features

* ⚡ Ultra Fast Performance (Astro SSG)
* 📱 Fully Responsive Design
* 🎯 SEO Optimized Structure
* 🎨 Clean & Modern UI

## 🧠 Insight Teknis

Astro digunakan karena:

* Rendering HTML static → performa tinggi
* JavaScript minimal → hemat resource
* Hybrid support → bisa inject React/Vue

Cocok untuk environment dengan keterbatasan resource (RAM/VRAM).
